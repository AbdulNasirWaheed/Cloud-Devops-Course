📌 Day 14 — Deploying a Web App on AWS | DevOps & Cloud Program @MiseAcademy

Today marked a significant milestone: deploying a real web application on AWS EC2, accessible to anyone on the internet.

Key steps covered:

🔹 Provisioned an EC2 Ubuntu instance (Free Tier)
🔹 Connected via SSH using a key pair (.pem)
🔹 Installed and verified Nginx as the web server
🔹 Cloned the web app from GitHub onto the server
🔹 Deployed files to /var/www/html — Nginx's default serving directory
🔹 Configured Security Group inbound rules to allow HTTP (port 80)
🔹 Accessed the live site via the EC2 public IP

Key insight: Nginx doesn't just serve files — it handles load balancing, reverse proxying, and static content delivery at scale. Understanding it is fundamental for any Cloud or DevOps role.
