# Day-13: AWS EC2 Launch & SSH Connections

**#100DaysOfDevOps | DevOps & Cloud Journey**

## 📌 Objective
Provision the first AWS EC2 Ubuntu instance and establish secure SSH access from a local machine.

## 🏭 Industry Scenario
A DevOps Engineer must spin up a fresh Linux server and connect to it remotely using secure shell (SSH) to begin configuration and deployment tasks.

## ✅ What I Practiced
- Launching an Ubuntu EC2 instance via AWS Console
- Configuring Security Groups (inbound rules for SSH)
- Downloading and securing the `.pem` key pair
- Connecting to EC2 via SSH from Windows (Git Bash / PowerShell)
