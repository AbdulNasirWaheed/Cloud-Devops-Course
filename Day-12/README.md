# Day-12: AWS Account Provisioning

**#100DaysOfDevOps | DevOps & Cloud Journey**

## 📌 Objective
Set up the AWS cloud foundation following security best practices: account provisioning, billing alerts, and root account MFA configuration.

## 🏭 Industry Scenario
Starting a new cloud engineering role requires a properly secured AWS account before any infrastructure can be provisioned. A misconfigured root account is the most common cause of cloud security incidents.

## ✅ What I Practiced
- AWS account creation and initial console setup
- Configuring AWS Budgets and billing alerts
- Enabling Multi-Factor Authentication (MFA) on the root account
- Reviewing IAM Console best practices
